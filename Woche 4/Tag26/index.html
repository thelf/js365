Aufgabe Arbeiten mit der String-API - substr

Der Name des Anwenders besteht aus einem Vornamen und einen Nachnamen.

"use strict";

let name = "Ladislaus Jones";
let spacePosition = name.indexOf(" ");

// Die Zählung beginnt bei 0 (L von Ladislaus, von dort 9 Zeichen nach rechts)
let firstName = name.substr(0, spacePosition);

// Für den Nachnamen werden alle Buchstaben ab Position 10 benötigt. Auf das 2te Argumment kann verzichtet werden.
let lastName = name.substr(spacePosition +1);

console.log(firstName);
console.log(lastName);

In der Konsole wird der Vorname und der Nachname ausgegeben. Wenn der Benutzer zwei Vornamen und einen Nachnamen hat, ordnet das Programm den zweiten Vornamen dem Nachnamen zu.
