Übung Tag 10

Aufgabe

    1. Fragen Sie den Benutzer nach seinem Namen.
    2. Speichern Sie den Namen in eine Variable.
    3. Begrüßen Sie den Benutzer mittels einer Alert-Box.
    1. Stellen Sie dem Benutzer die Frage, ob er bereit ist zu beginnen.
    2. Als Antwortmöglichkeit sollen "Ja" und "Nein" in der Frage enthalten sein.
    3. Die Antwort soll in einer weiteren Variable gespeichert werden.
    4. Erstellen Sie anschließend eine if-Verzweigung. Der Rumpf soll nur ausgewertet werden, wenn die Antwort "Ja" lautet.
    5. In dem Rumpf soll außerdem ein alert ausgegeben werden, mit der Antwort des Benutzers.
    1. Erstellen Sie eine weitere if-Verzweigung, falls der Benutzer mit "Nein" antwortet
    2. Die Antwort des Benutzers soll in einer weiteren Alert-Box ausgegeben werden.
    1. Deklarieren Sie eine neue Variable, darin soll die Antwort des Benutzers auf folgenede Frage gespeichert sein: "Welcher Operator hat die höhere Priorität, * oder +?"
    2. Um auf die Antwort reagieren zu können, schreiben Sie eine weitere if-Verzweigung, deren Rumpf nur ausgewertet wird, wenn die Antwort stimmt.
