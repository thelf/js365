Typenkonvertierung

Bestimmen Sie Rückgabewert und Datentyp folgender Ausdrücke:

  1. "1.5" * 2
    Rückgabewert: 3
    Datentyp: number
  2. "1,5" * 2
    Rückgabewert: NaN
    Datentyp: number
  3. "1.5" + 2
    Rückgabewert: "1.52"
    Datentyp: string
  4. Number("1.5") * 2
    Rückgabewert: 3
    Datentyp: number
  5. Number("1,5") * 2
    Rückgabewert: NaN
    Datentyp: number
  6. Number("3 Tage") * 7
    Rückgabewert: NaN
    Datentyp: number
  7. Number("Seite 20") + 5
    Rückgabewert: NaN
    Datentyp: number
  8. "9,2" + Number("11.7");
    Rückgabewert: 9,211.7
    Datentyp: string
  9. (NaN - 2) * (4/2)
    Rückgabewert: NaN
    Datentyp: number
  10. alert(Number(17 / 2 + 1.3))
    Rückgabewert: undefined
    Datentyp: undefined
  11. typeof 12.25
    Rückgabewert: "number"
    Datentyp: string
  12. typeof typeof 12.25
    Rückgabewert: "string"
    Datentyp: string
