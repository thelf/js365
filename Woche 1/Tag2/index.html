Aufgabe

Kraftstoff Verbrauch für PKW berechnen

  1. Der Benutzer soll die gefahrenen Kilometer (Strecke) eingeben und die verbrauchte Benzinmenge
  2. Das Programm ermittelt den Kraftstoffverbrauch auf 100 km und gibt ihn aus
  3. Der Benutzer soll nach seiner Tankgröße gefragt werden
  4. Ergänzend soll die maximale Reichweite in Kilometern berechnet werden

Folgende Formeln sollen verwendet werden:

  • Verbrauch [l/100km] = Benzinmenge [l] / Strecke [km] x 100
  • Reichweite [km] = Tankgröße [l] / Verbrauch [l/km]

Lösung

"use strict";

let strecke = Number(prompt( "Bitte geben Sie die gefahrenen Kilometer ein." ));
let liter = Number(prompt( "Bitte geben Sie nun die verbrauchte Benzinmenge ein." ));
let tankGroesse = Number(prompt( "Bitte geben Sie abschließend auch noch Ihre Tankgröße an." ));

const KM_ANGABE = 100

let verbrauch = (liter / strecke) * KM_ANGABE;
let reichweite = tankGroesse * (strecke / verbrauch);

alert( `Der Kraftstoffverbrauch auf 100 km beträgt ${verbrauch} Liter. 
Ihre maximale Reichweite bei einer Tankgröße von ${tankGroesse} Litern beträgt ${reichweite} Kilometer:` );
